HTML语法使用说明

1. HTML基础结构

HTML文档的基本结构包括以下部分：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>网页标题</title>
</head>
<body>
    <!-- 内容写在这里 -->
</body>
</html>

说明：<!DOCTYPE html>声明HTML5文档，<html>是根元素，<head>包含元数据，<body>包含网页可见内容。

2. 常用HTML标签

以下是常用的HTML标签及其用途：

<h1>至<h6>：标题（从大到小）
<p>：段落
<a href="URL">：超链接
<img src="图片路径" alt="描述">：图片
<imgs src="img src="图片路径" alt="描述" width="300" height="200"">：图片2
<div>：块级容器
<span>：内联容器
<it>:notes                
<ul>、<ol>、<li>：无序/有序列表
<table>、<tr>、<td>：表格

示例：

<h1>主标题</h1>
<p>这是一个段落。</p>
<a href="https://example.com">点击跳转</a>
3. HTML属性

属性为标签提供额外信息，写在开始标签内，格式为 name="value"。

<a href="https://example.com" target="_blank">打开新窗口</a>
<img src="image.jpg" alt="图片描述" width="200">

常用属性：

  • id：唯一标识符
  • class：样式类
  • style：内联CSS
  • title：提示文本

4. 嵌入CSS样式

可以在<head>或<style>标签中添加CSS，也可以使用内联样式。

<style>
    h1 { color: blue; }
    .my-class { font-size: 16px; }
</style>
<!-- 或内联样式 -->
<p style="color: red;">红色文本</p>
5. 嵌入JavaScript

使用<script>标签嵌入JavaScript代码，通常放在<body>底部或<head>内。

<script>
    alert('Hello, World!');
</script>

注意：复杂的JavaScript逻辑建议放在外部文件，通过<script src="file.js">引入。

6. HTML注释

使用<!-- 注释内容 -->添加注释，注释不会显示在页面上。

<!-- 这是一个注释 -->
7. 常见问题与提示

  • 确保标签正确嵌套，避免交叉嵌套（如<p><span></p></span>）。
  • 属性值需用引号包裹（如href="URL"）。
  • 图片路径需正确，建议使用相对路径或完整URL。
  • 预览中的JavaScript可能受iframe安全限制影响，复杂功能需导出后测试。
